अत्यन्तशुद्धचिन्मात्रे परिणामश्चिराय यः ।
तुर्यातीतं पदं तत्स्यात्तत्स्थो भूयो न शोचति ॥
Padaccheda & Samasa Analysis
अत्यन्तशुद्धचिन्मात्रे अत्यन्त-शुद्ध-चित्-मात्रे
In the extremely pure absolute consciousness
Samasa अत्यन्तं शुद्धं च तत् चिन्मात्रं (कर्मधारयः)
परिणामः परिणामः
Transformation / State (of absorption)
चिराय चिराय
For a long time
यः यः
Which
तुर्यातीतम् तुर्य-अतीतम्
Beyond the Fourth State (Turiya)
Samasa तुर्यम् अतीतम् (द्वितीया-तत्पुरुषः)
पदम् पदम्
Position / State
तत् तत्
That
स्यात् स्यात्
Is / Would be
तत्स्थः तत्-स्थः
Established in that
Samasa तत्र तिष्ठति इति (उपपद-तत्पुरुषः)
भूयः भूयः
Again / Anymore
न न
Not
शोचति शोचति
Grieves / Broods
Translation
"In that extremely pure absolute consciousness, the state (of absorption) which lasts for a long time—that is the position beyond the Fourth (Turya-atita, or Brahman). One established in that state does not grieve anymore."
