अतिरमणीये काव्ये पिशुनोऽन्वेषयति दूषणान्येव।
अतिरमणीये वपुषि व्रणमेव हि मक्षिकानिकरः॥
Padaccheda & Samasa Analysis
अतिरमणीये अति-रमणीये
In an extremely beautiful (poem/body)
Samasa Vigraha अत्यन्तं रमणीयम् = अतिरमणीयम् (प्रादितत्पुरुषः)
काव्ये काव्ये
In a poem
पिशुनः पिशुनः
A wicked person / backbiter / slanderer
अन्वेषयति अन्वेषयति
Searches for / seeks
दूषणानि दूषणानि
Flaws / faults / defects
एव एव (अव्ययम्)
Only / indeed
वपुषि वपुषि
In a body
व्रणम् व्रणम्
A wound / sore
हि हि (अव्ययम्)
Surely / because
मक्षिकानिकरः मक्षिका-निकरः
A swarm of flies / a collection of flies
Samasa Vigraha मक्षिकाणां निकरः (षष्ठी-तत्पुरुषः)
Translations
"A complainant seeks only flaws in a very beautiful poem. Indeed, a swarm of flies surround only a wound on a very beautiful body."
